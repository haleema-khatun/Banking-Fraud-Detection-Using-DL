Enhancing Fraud Detection in Banking Transactions Using Deep Learning

About the System

This application is designed to detect potentially fraudulent banking transactions using advanced machine learning techniques. The system provides a secure and accurate platform for both administrators and users to analyze financial transaction data.

The administrator is responsible for uploading the dataset, splitting the data, training multiple machine learning models, and comparing their performance using visual graphs. The best-performing model is automatically selected and saved for prediction.

Users can test individual transactions in real time or upload complete CSV files for bulk fraud detection. Each transaction is classified as either: Fraud (1) or Normal (0).
